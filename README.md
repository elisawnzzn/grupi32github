# Menaxhimi i Studentëve – C++

Ky projekt është një aplikacion i thjeshtë në **C++** që menaxhon të dhënat e një studenti, llogarit mesataren e notave dhe përcakton nëse studenti kalon apo dështon bazuar në pjesëmarrje dhe rezultate.

---

# Funksionalitetet
- Regjistrimi i studentit (emër, ID, semestër)
- Futja e pjesëmarrjes dhe notave
- Llogaritja e mesatares
- Vendimi: **kalon / dështon**
- Gjenerimi i raportit përfundimtar

---

g++ main.cpp -o studenti
